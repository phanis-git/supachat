Prometheus
----------
Step 4 — Run Queries in Prometheus
Click "Graph" tab at top. Then try these queries:
Query 1 — See all metrics available:
http_requests_total
Click "Execute" → shows all request counts
Query 2 — Request rate per second:
rate(http_requests_total[5m])
Query 3 — Average latency:
rate(http_request_duration_seconds_sum[5m])
/
rate(http_request_duration_seconds_count[5m])
Query 4 — Check backend is up:
up{job="backend"}
Returns 1 = UP ✅, 0 = DOWN ❌
Query 5 — Total requests count:
http_requests_total
